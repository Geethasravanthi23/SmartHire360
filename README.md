# SmartHire360

SmartHire360 is a Full Stack Job Tracking and Career Management Platform built using Java, Spring Boot, React.js, and MySQL.

## Features

- User Registration & Login
- Profile Management
- Job Listings
- Application Tracking
- Resume Analyzer
- ATS Score Calculation
- Job Match Analyzer
- Skill Suggestions
- Dashboard Analytics
- Export Applications to Excel & PDF

## Tech Stack

### Frontend
- React.js
- Vite
- Axios
- Recharts

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Spring Security

### Database
- MySQL

## Project Structure

SmartHire360
├── smarthire360 (Spring Boot Backend)
└── smarthire360-frontend (React Frontend)

### Backend

```bash
cd smarthire360
mvn spring-boot:run
```

### Frontend

```bash
cd smarthire360-frontend
npm install
npm run dev
```
